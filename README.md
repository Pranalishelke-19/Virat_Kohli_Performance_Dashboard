# Virat_Kohli_Performance_Dashboard

## 📊 Project Overview
This project analyzes the performance of Virat Kohli using Power BI.  
The dashboard provides insights into batting performance across different years and formats.

---

## 🛠 Tools Used
- Power BI  
- Excel / CSV Dataset  

---

## 📁 Dataset Features
The dataset includes the following columns:
- Runs  
- Innings  
- Outs  
- Highest Score (HS)  
- Average (Avg)  
- Strike Rate (SR)  
- Balls Faced  
- 100s (Centuries)  
- 50s (Half-centuries)  
- 4s (Fours)  
- 6s (Sixes)  
- Dot Ball %  
- Format (Test, ODI, T20)  
- Year  

---

## 📈 Dashboard Insights
- Year-wise performance analysis  
- Format-wise comparison (Test, ODI, T20)  
- Total runs and strike rate trends  
- Boundary analysis (4s and 6s)  
- Century and half-century distribution  
- Dot ball percentage analysis  

---

## 🎯 Key Highlights
- Identified peak performance years  
- Compared performance across formats  
- Analyzed scoring patterns and consistency  

---

## 📌 Conclusion
This dashboard helps to understand Virat Kohli’s batting performance using visual insights and key metrics.
